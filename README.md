# 🦋 Bonna's Space — Premium Retro RPG Portfolio

Welcome to the digital corner of **Bonna**, a digital artist who loves drawing OCs, fanarts, and all things pixel! This portfolio is designed as an immersive "Retro RPG" experience, blending neo-brutalist aesthetics with modern interactivity.

## 🕹️ Key Features

- **Seamless Transitions**: A custom dynamic page loader ensures the journey between `Home`, `Commission`, and `ToS` is smooth and instantaneous without a single browser refresh.
- **Persistent RPG Soundscape**: A lo-fi background track that never stops, complemented by tactile "pixel-blip" sound effects for every interaction.
- **Responsive 3D Interactivity**: Bento-style cards with normalized 3D tilt effects and parallax pixel-art mascots that follow your journey.
- **Performance First**: Built with a "Low Tech, High Aesthetic" philosophy—using only Vanilla HTML, CSS, and JS for maximum speed and accessibility.

## 🛠️ Built With

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS (CSS Variables, Flex/Grid, Custom Animations)
- **Logic**: Vanilla JavaScript (SPA Engine, Audio Manager, Intersection Observer)
- **Aesthetics**: Google Fonts (Press Start 2P, VT323), FontAwesome, & Pixel Art Assets.
